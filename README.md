AncestryDesert.ipynb – Jupyter notebook to visualize genomic “desert” regions with unusually low archaic ancestry across human chromosomes.

OmicronMutantAnalysis.ipynb – Jupyter notebook analyzing SARS-CoV-2 Omicron mutations, looking at effects and potential for epistasis.

bam_functions.py – Utility module with helper functions for working with BAM alignment files.

isoform_correlations_binned_expression.py – Script to bin expression levels and visualize transcript isoforms biotypes across expression levels.

kallisto_isoform_proportion.py – Script that processes Kallisto quantification output to calculate isoform proportions per gene and generate summary tables for downstream analysis.

manual_annotation_pipeline.nf – Nextflow pipeline that allows raw read input to annotated isoform output.

remove_overlapping_gff_features.sh – Shell script to clean GFF annotation files by removing or resolving overlapping features based on simple coordinate rules.
